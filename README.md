# Simple C Kernel for Jupyter (Windows)

**Windows 환경의 Jupyter Notebook에서 C언어를 실행할 수 있는 커널입니다.**

기존 주피터 C 커널들은 `scanf`를 만나면 멈추거나 입력창이 뜨지 않는 문제가 있었습니다.
이 커널은 **네이티브 입력창**을 지원하여, 터미널에서 실행하는 것과 똑같은 경험을 제공합니다.

---

## 설치 가이드

이 커널을 사용하려면 **Python**과 **GCC 컴파일러**가 필요합니다.

### 1단계: Python 설치 확인
1.  키보드의 `Win`키 + `R`을 누르고 `cmd`를 입력해 터미널을 엽니다.
2.  아래 명령어를 입력하세요.
    ```bash
    python --version
    ```
3.  버전이 나온다면 통과! 에러가 난다면 [Python 공식 홈페이지](https://www.python.org/downloads/)에서 다운로드하여 설치하세요.
    * **주의:** 설치할 때 **"Add Python to PATH"** 체크박스를 반드시 선택해야 합니다!

### 2단계: GCC 컴파일러 설치 (가장 중요 ⭐)
C언어를 번역해줄 번역기(컴파일러)가 필요합니다. **MinGW-w64**를 설치합니다.

1.  **[WinLibs 다운로드 링크](https://winlibs.com/#download-release)** 로 이동합니다.
2.  최신 버전의 **Zip archive** 을 다운로드합니다.
3.  다운로드한 압축 파일을 `C:\` 드라이브 최상단에 풉니다.
    * 예시 경로: `C:\mingw64`
4.  **환경 변수 설정:**
    * 윈도우 검색창에 **"시스템 환경 변수 편집"**을 검색하고 실행합니다.
    * **[환경 변수]** 버튼 클릭 -> **[시스템 변수]** 목록에서 **`Path`**를 찾아 더블 클릭합니다.
    * **[새로 만들기]** 를 누르고, 아까 압축 푼 폴더의 `bin` 폴더 경로를 붙여넣습니다.
        * 예: `C:\mingw64\bin`
    * 확인, 확인을 눌러 창을 닫습니다.
5.  설치 확인:
    * 새 `cmd` 창을 열고 `gcc --version`을 입력했을 때 버전 정보가 나오면 성공!

### 3단계: 커널 설치
1.  이 저장소(폴더)를 다운로드하고 압축을 풉니다.
2.  폴더 안에서 터미널(CMD)을 열고 아래 명령어를 입력합니다.
    ```bash
    python install.py
    ```
3.  "✅ 설치가 완료되었습니다!" 메시지가 나오면 끝입니다.

---

## 사용 방법

1.  Jupyter Notebook을 실행합니다.
2.  우측 상단 **New** 버튼을 누르고 **Simple C Kernel**을 선택합니다.
3.  이제 C 코드를 작성하고 실행(`Shift + Enter`)하세요!
